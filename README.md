# MERN Item Manager (WMT Lab Test Pre-Practice)

## 📌 Project Description

This is a pre-practice MERN stack project created for WMT lab test preparation.
The system is currently under development and not fully integrated.

---

## ⚠️ Project Status

🚧 This project is **not fully completed**

* Frontend and backend are partially implemented
* Some features may not work correctly
* Integration is still in progress

---

## 🚀 Planned Features

* Add items
* View items
* Update items
* Delete items

---

## 🛠️ Technologies Used

* MongoDB
* Express.js
* React.js
* Node.js

---

## ⚙️ Setup Instructions

### Install dependencies

Backend:
cd backend
npm install

Frontend:
cd frontend
npm install

---

## ▶️ Run the Project

Start backend:
cd backend
npm start

Start frontend:
cd frontend
npm start

---

## 🧪 Testing

APIs can be tested using Postman (some endpoints may not work yet).
